# Context

A single-file PWA the author opens as a **new-tab page**: each load pulls one longevity
habit into focus as a health reminder. A personal tool, built for daily use.

## Glossary

**Reminder** — one longevity **Habit** pulled into focus on a new tab. The unit the app
exists to surface; "pick another" swaps the current Reminder for a different one.

**Habit** — one of the 42 lines quoted verbatim from Bryan Johnson's public X post. Quoted
for personal reference and commentary, not owned by this project.

**Personal project** — the project's framing. Built for the author's own daily use.
Attribution (name, avatar, handle, quoted text) and the not-affiliated disclaimer are kept
to credit the source of the Habits, not to imply association.

**Attribution** — Bryan Johnson's name, @handle, avatar, and verified badge shown in the
app. Present to credit the source of the quoted Habits, not to imply association. Always
paired with the not-affiliated disclaimer.
